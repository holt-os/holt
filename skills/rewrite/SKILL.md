# rewrite

## When to use

The user pastes a paragraph, email, or passage and wants it cleaner and clearer while keeping the same meaning. This is a neutral clarity pass, distinct from a voice-anchored writing task: keep the author's own voice unless a tone is requested.

## Instructions

1. Preserve intent and every fact. Do not add claims, drop caveats, or change the point being made.
2. Cut filler and hedging: "just", "really", "I think that", "in order to", throat-clearing intros, and redundant phrases.
3. Prefer plain words over inflated ones ("use" not "utilize", "help" not "facilitate").
4. Vary sentence length so it reads naturally; break up run-ons, and do not make everything staccato.
5. Keep the author's voice and register. Only shift tone (more formal, warmer, punchier) if the user asks for it.
6. Return only the rewritten text, then one short line noting what you changed (for example: "tightened, cut hedging, split two long sentences").

## Example

Before: I just wanted to reach out in order to see if you might possibly have some availability to maybe hop on a quick call sometime.

After: Do you have time for a quick call this week?

Changed: cut hedging and filler, made it a direct ask.
